# Metabolic dysregulation and knee pain study

This repository contains the analytical code used for cross-sectional and longitudinal analyses of metabolic dysregulation and knee pain.

## Contents

* Org-mode (Babel) workflow
* R scripts for statistical analysis

## Data availability

Data are not publicly available due to institutional regulations governing access to individual-level data.

## Reproducibility

The code reflects the original analytical workflow used in the study and may require adaptation for use with other datasets.
